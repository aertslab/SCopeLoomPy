# SCopeLoomPy v0.1.0
A Python package to create .loom files and extend them with other data e.g.: SCENIC regulons, Seurat clusters and markers, compatible with SCope.

## Installation
```
pip install https://github.com/aertslab/SCopeLoomPy.git
```
